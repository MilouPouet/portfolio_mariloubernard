FICHE TECHNIQUE — Portfolio Web

Objectif du projet :
Créer un portfolio personnel afin de présenter mon identité, mes compétences, mon expérience et mes informations de contact. Le site servira également de support pour ma recherche de stage.

Structure du site :
- Accueil (Hero) : Nom + objectif
- À propos : Photo + présentation
- Loisirs : 3 cartes interactives avec images
- Expérience professionnelle
- Formation
- Informations complémentaires (langues et handicap)
- Contact
- Footer

Choix graphiques :
Palette : Dégradé (rose, violet, orange) style Instagram
Police : Roboto
Mise en page : Flexbox
Effets : Hover zoom, ombres, sections plein écran

Responsive Design :
- Flex-wrap + media queries
- Design adaptable sur tablette et ordinateur

Accessibilité :
- Contraste texte/fond suffisant
- Présence d’attributs alt
- Points d’amélioration : cohérence titres

Points à améliorer :
- Améliorer la version mobile
- Optimiser la sémantique HTML

Résultat :
Le projet est fonctionnel, lisible, moderne et évolutif. Il constitue une première version professionnelle de portfolio web.
